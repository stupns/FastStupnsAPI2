# Firewall

```commandline
sudo ufw status
sudo ufw allow http
sudo ufw allow https
sudo ufw allow ssh
sudo ufw allow 5432

sudo ufw enable
sudo ufw status
```

> __Note:__  If u want delete notification, use **sudo ufw delete allow 5432**
